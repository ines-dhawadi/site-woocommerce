== Description ==
Description: Relic Fashion Store is a modern free WooCommerce WordPress theme especially designed for shops and eCommerce sites. It could be used for all types of ecommerce online stores, including: Fashion, Shoes, Jewelry, Watch, Sport, Technology, Mobile... Relic Fashion Store is easy to use and fully responsive.Theme is translation and multilinual ready. Relic Fashion Store is SEO friendly helping you to achieve the highest positions on Google and optimized codebase to maximize your siteâ€™s performance and loading speed.  Main theme features: 2 menu positions, advance search bar,top Header, shop by category, full width owl carousel with custom products, left or right sidebars,both sidebar, footer widget area, custom logo, custom background, breadcrumbs, related posts, one homepage styles, fast loading, cross-browser compatibility, font awesome, documentation , support, bootstrap grid layout. Theme including support for grid/list toggle shop page,wishlist, quick view and compare plugins . SEO Friendly.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
